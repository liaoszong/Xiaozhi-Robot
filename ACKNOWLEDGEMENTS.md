# Acknowledgements

本项目 ESP32 侧基于开源项目 [78/xiaozhi-esp32](https://github.com/78/xiaozhi-esp32) 进行二次开发。感谢原项目作者、维护者和社区提供的小智 ESP32 语音交互、设备端框架、音频处理、显示和 MCP 工具机制等基础能力。

本项目主要在上游能力基础上完成与 STM32F103RCT6 小车底盘控制系统的集成，扩展了语音控车、UART 安全帧、动作序列和任务场景演示等功能。

同时感谢以下工具链和开发框架对项目开发的支持：

- ESP-IDF；
- STM32CubeMX / STM32 HAL；
- Keil MDK / STM32CubeIDE；
- Git / CMake / Ninja 等工程工具。

本项目仅作为大学生创新训练和嵌入式系统学习实践使用，相关开源代码遵循其原项目许可证要求。
